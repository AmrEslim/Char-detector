\doxysubsection{Detailed Description}
Responsible for drawing the grid of a \doxylink{class_q_c_p_axis}{QCPAxis}. 

This class is tightly bound to \doxylink{class_q_c_p_axis}{QCPAxis}. Every axis owns a grid instance and uses it to draw the grid lines, sub grid lines and zero-\/line. You can interact with the grid of an axis via \doxylink{class_q_c_p_axis_a63f1dd2df663680d2a8d06c19592dd63}{QCPAxis\+::grid}. Normally, you don\textquotesingle{}t need to create an instance of \doxylink{class_q_c_p_grid}{QCPGrid} yourself.

The axis and grid drawing was split into two classes to allow them to be placed on different layers (both \doxylink{class_q_c_p_axis}{QCPAxis} and \doxylink{class_q_c_p_grid}{QCPGrid} inherit from \doxylink{class_q_c_p_layerable}{QCPLayerable}). Thus it is possible to have the grid in the background and the axes in the foreground, and any plottables/items in between. This described situation is the default setup, see the \doxylink{class_q_c_p_layer}{QCPLayer} documentation.

{\footnotesize\ttfamily void QCPGrid\+::set\+Zero\+Line\+Pen (\begin{DoxyParamCaption}\item[{const QPen \&}]{pen }\end{DoxyParamCaption})}

Sets the pen with which zero lines are drawn.

Zero lines are lines at value coordinate 0 which may be drawn with a different pen than other grid lines. To disable zero lines and just draw normal grid lines at zero, set {\itshape pen} to Qt\+::\+No\+Pen.
