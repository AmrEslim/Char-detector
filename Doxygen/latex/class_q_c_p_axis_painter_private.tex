\doxysubsection{Detailed Description}
(Private) 

This is a private class and not part of the public \doxylink{class_q_custom_plot}{QCustom\+Plot} interface.

It is used by \doxylink{class_q_c_p_axis}{QCPAxis} to do the low-\/level drawing of axis backbone, tick marks, tick labels and axis label. It also buffers the labels to reduce replot times. The parameters are configured by directly accessing the public member variables.

Draws a single tick label with the provided {\itshape painter}, utilizing the internal label cache to significantly speed up drawing of labels that were drawn in previous calls. The tick label is always bound to an axis, the distance to the axis is controllable via {\itshape distance\+To\+Axis} in pixels. The pixel position in the axis direction is passed in the {\itshape position} parameter. Hence for the bottom axis, {\itshape position} would indicate the horizontal pixel position (not coordinate), at which the label should be drawn.

In order to later draw the axis label in a place that doesn\textquotesingle{}t overlap with the tick labels, the largest tick label size is needed. This is acquired by passing a {\itshape tick\+Labels\+Size} to the \doxylink{class_q_c_p_axis_painter_private_aab666e1d85b234afbc88076c7ba9424a}{draw\+Tick\+Label} calls during the process of drawing all tick labels of one axis. In every call, {\itshape tick\+Labels\+Size} is expanded, if the drawn label exceeds the value {\itshape tick\+Labels\+Size} currently holds.

The label is drawn with the font and pen that are currently set on the {\itshape painter}. To draw superscripted powers, the font is temporarily made smaller by a fixed factor (see \doxylink{class_q_c_p_axis_painter_private_a5331f78e398da5442767c261578da502}{get\+Tick\+Label\+Data}). \Hypertarget{class_q_c_p_axis_painter_private_a62aa9247b2ab22a3e069b9b867376f85}\label{class_q_c_p_axis_painter_private_a62aa9247b2ab22a3e069b9b867376f85}
