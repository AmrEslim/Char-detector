\doxysubsection{Detailed Description}
A plottable representing a parametric curve in a plot. 



Unlike \doxylink{class_q_c_p_graph}{QCPGraph}, plottables of this type may have multiple points with the same key coordinate, so their visual representation can have {\itshape loops}. This is realized by introducing a third coordinate {\itshape t}, which defines the order of the points described by the other two coordinates {\itshape x} and {\itshape y}.

To plot data, assign it with the \doxylink{class_q_c_p_curve_a41246850d2e080bc57183ca19cd4135e}{set\+Data} or \doxylink{class_q_c_p_curve_a73edf394b94f3f24f07518e30565a07f}{add\+Data} functions. Alternatively, you can also access and modify the curve\textquotesingle{}s data via the \doxylink{class_q_c_p_curve_a761492fd00b1ab7cb18ce23c118c6c60}{data} method, which returns a pointer to the internal QCPCurve\+Data\+Container.

Gaps in the curve can be created by adding data points with NaN as key and value ({\ttfamily q\+QNa\+N()} or {\ttfamily std\+::numeric\+\_\+limits$<$double$>$\+::quiet\+\_\+\+Na\+N()}) in between the two data points that shall be separated.\hypertarget{class_q_c_p_curve_qcpcurve-appearance}{}\doxysubsection{\texorpdfstring{Changing the appearance}{Changing the appearance}}\label{class_q_c_p_curve_qcpcurve-appearance}
The appearance of the curve is determined by the pen and the brush (\doxylink{class_q_c_p_abstract_plottable_ab74b09ae4c0e7e13142fe4b5bf46cac7}{set\+Pen}, \doxylink{class_q_c_p_abstract_plottable_a7a4b92144dca6453a1f0f210e27edc74}{set\+Brush}).\hypertarget{class_q_c_p_curve_qcpcurve-usage}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{class_q_c_p_curve_qcpcurve-usage}
Like all data representing objects in \doxylink{class_q_custom_plot}{QCustom\+Plot}, the \doxylink{class_q_c_p_curve}{QCPCurve} is a plottable (\doxylink{class_q_c_p_abstract_plottable}{QCPAbstract\+Plottable}). So the plottable-\/interface of \doxylink{class_q_custom_plot}{QCustom\+Plot} applies (\doxylink{class_q_custom_plot_a32de81ff53e263e785b83b52ecd99d6f}{QCustom\+Plot\+::plottable}, \doxylink{class_q_custom_plot_af3dafd56884208474f311d6226513ab2}{QCustom\+Plot\+::remove\+Plottable}, etc.)
